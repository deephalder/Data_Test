This analysis attempts to predicate the probability for survival of the
Titanic passengers. In order to do this, I will use the different
features available about the passengers, use a subset of the data to
train an algorithm and then run the algorithm on the rest of the data
set to get a prediction.

In this analysis I have done the following :

  Optimized the imported data set and handled the missing values

  passenger's chance of survival.
\end{enumerate}
